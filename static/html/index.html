OpinionLens

A production-ready sentiment analysis project leveraging local ML training with tracked experiments, DVC data versioning and pipelines, remote MLflow model registry, Dockerized services, and Prometheus/Grafana monitoring with inference instrumentation.

Available API Endpoints

Try the following available API endpoints for programmatic sentiment analysis.

Method URL Parameters Description
GET /api/v1/inference/predict text (str: query parameter) Quick analysis of a single text
POST /api/v1/inference/predict text (str: body parameter) Secure analysis of a single text
POST /api/v1/inference/batch_predict (list[str]: body parameter) Quick and secure analysis of multiple texts
